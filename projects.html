🎙️ VANI
IoT Project
Voice Revenue Smart Living

VANI is a smart home automation system I built as part of my IoT coursework at REVA University. The system uses an ESP32 microcontroller connected to a custom PCB relay board, a DHT11 temperature and humidity sensor, and a BLE Controller mobile app for wireless control.

The project enables users to control home appliances (lights, fans, other devices) wirelessly over Bluetooth Low Energy (BLE) directly from their smartphone — no internet connection required. The DHT11 sensor provides real-time temperature and humidity readings displayed in the app.

This project taught me the full IoT pipeline — from hardware circuit design and PCB layout to firmware programming on the ESP32 and testing end-to-end BLE communication.

🚨 RapidReach
Startup Venture IGNITE Program
Hyperlocal Emergency Response Platform

RapidReach is a browser-based emergency response platform I co-built as part of the Wadhwani Foundation IGNITE Program — an innovation and entrepreneurship initiative. I served as the UI/UX Designer and Product Planner on a four-person team building this venture.

The platform is designed for urban and semi-urban India, connecting people in emergencies with the nearest available volunteers, responders, and resources — faster than traditional emergency systems. The platform features an SOS Dashboard, Volunteer Portal, Admin Panel, and Registration/Login flows.

My responsibilities included designing all five key website pages (Landing Page, SOS Dashboard, Volunteer Portal, Register/Login, Admin Panel), creating user journey flowcharts, defining core product features, and producing formal deliverable documents for the IGNITE program milestones.

A key moment in this project was a strategic pivot — shifting from a mobile app to a browser-based website to maximize accessibility across devices without requiring downloads.
